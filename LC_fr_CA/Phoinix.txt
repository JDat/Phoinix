Phoinix.txt
===========

Qu'est-ce que Phoinix?
----------------------

Phoinix ['foiniks] est le mot grec pour palmier dattier. C'est aussi
le nom d'un oiseau mythique venant de l'Egypte qui selon la légende se
brûla de par lui-même et renît de ses cendres après 500 ans.

Dans ce contexte Phoinix est le nom de l'émulateur Nintendo
Gameboy(TM) pour la plateforme Palm OS. On l'a autrefois connu comme
PalmBoy, mais Palm Inc. fît une réclamations de cette marque déposée.

Autre langage
-------------

La langue standard de Phoinix est l'anglais. D'autres langues sont
supporté dans l'application et le Lisez-moi. Tous les autres textes
sont actuellement en anglais, désolé.

Démarage rapide
---------------

SVP, visité le site web http://phoinix.sourceforge.net tel que
mentionné dans le fichier d'information de votre application. On y
retrouve de nombreuse information pertinente.

Sélectionné le fichier de votre langage, décompresser le Si vous lisez
ce fichier, vous l'avez probablement déjà fait) Installez alors le
fichier "Phoinix.prc".

Pour vérifier si tout fonctionne convenablement, il y a quelques jeux
libres dans les archives "freegame.zip". Ces jeux sont déjà convertit,
ils suffit de les décompressé et de les installés. Ils sont déjà
convertis, juste s'ouvrent et installent comme avant.

Amusez-vous...

Par la suite, vous pourriez souhaiter lire la documentation. Pour
faciliter la maintenance, j'ai compressé les fichier web dans
"doc.zip". Utilisé n'imporqu'elle navigateur web ou éditeur texte pour
lire la documentation.
